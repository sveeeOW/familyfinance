  инлайн-фолбэк без Redis; async-эндпоинт `POST /ai/recognize-async` (202)

## Как проверить
1. `docker compose up -d` (Postgres+Redis+backend) → миграции → `npm run seed`.
2. Открыть `http://localhost:3000/docs` — все эндпоинты.
3. Установить `AI_PROVIDER=mock` для работы без ключа Anthropic (распознавание текста по правилам).
4. Прописать `TELEGRAM_BOT_TOKEN` для запуска бота (long polling).

### Автоматическая end-to-end проверка
`cd apps/backend && npm run verify:e2e` — поднимает встроенный PostgreSQL (без Docker),
применяет схему, сидит категории, запускает реальное Nest-приложение и прогоняет основной
сценарий §31 по HTTP (регистрация → портфели → доход → расход → AI-распознавание → invite/accept →
split → аналитика → дедуп → push-токен → async-распознавание). **Статус: 27/27 проверок проходят.**

> На Windows запускать из обычной (не-administrator) консоли — PostgreSQL не стартует под
> правами администратора.
