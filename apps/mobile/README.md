# Family Finance — мобильное приложение (Expo / React Native)

Каркас мобильного клиента: авторизация, нижнее меню (§26.3), главный экран,
расходы (список + добавление), доходы, портфели (с invite-ссылкой), аналитика,
настройки с привязкой Telegram-бота.

## Запуск

```bash
npm install
cp .env.example .env          # EXPO_PUBLIC_API_URL — адрес backend
npm start                     # затем i (iOS) / a (Android) / w (web)
```

На реальном устройстве укажите в `.env` IP машины с backend (не `localhost`).

## Структура

```
src/
├── api/           — fetch-клиент с авто-refresh токена, типы, эндпоинты
├── store/         — Zustand: авторизация и выбранный портфель
├── navigation/    — стек авторизации + нижние табы
├── components/    — переиспользуемый UI (карточки, кнопки, поля, выбор портфеля)
├── screens/       — экраны приложения
└── theme.ts       — палитра (§26.1)
```

## Что доработать
- Графики (круговые/линейные) — сейчас простые бары.
- Экран уточнения расходов, экраны кредитов и инвестиций.
- Загрузка скриншота чека из приложения (есть API `/ai/recognize-expense`).
- Хранение токенов в `expo-secure-store` вместо AsyncStorage для продакшена.
